# Simple Minesweeper (VB.NET WinForms)

A beginner-friendly, resizable Minesweeper game built with VB.NET WinForms.  
This project features a 9×9 grid with 10 randomly placed mines. The UI automatically resizes with the window.

## 🧩 Features

- 9×9 grid with 10 mines
- Left-click to reveal, right-click to flag
- Auto-resizing buttons and font
- Win/loss detection
- Simple and clean codebase

## 🛠 Requirements

- Windows with .NET Framework 4.8+
- Visual Studio (recommended)

## 🚀 How to Run

1. Open the solution in Visual Studio.
2. Build and run the project.
3. Resize the window to see the grid adapt!

# シンプルマインスイーパー（VB.NET WinForms）

VB.NET WinForms で作られた、初心者向けのマインスイーパーゲームです。  
9×9 のマスにランダムで 10 個の地雷が配置され、ウィンドウサイズに応じて UI が自動調整されます。

## 🧩 特徴

- 9×9 グリッドに 10 個の地雷
- 左クリックで開く、右クリックでフラグ
- ウィンドウサイズに応じてボタンと文字サイズが自動調整
- 勝利・敗北の判定あり
- シンプルで読みやすいコード

## 🛠 必要環境

- .NET Framework 4.8 以上がインストールされた Windows
- Visual Studio（推奨）

## 🚀 実行方法

1. Visual Studio でソリューションを開く。
2. ビルドして実行。
3. ウィンドウサイズを変えて、マスの自動調整を体験しよう！
